Types os Selectors :
1.Global/Universal Selector
2.Tag/Element Selector
3.Id selector
4.Class Selector
5.Attritube selector

1.Global/Universal Selector:
global/universal selector is the form of selecting the entire data for the style,(*) is used to declare
2.Tag/Element Selector:
(Tag Name ) is used to declare the styles to it
3.Id Selector:
It is defined as the most effective selector,it uses (#) for declaring the style
4.Class Selector:
It uses (.) to declare the styles
5.Attribute Selector:
It uses ([]) for declaring the styles ex:[class="attribute name"
